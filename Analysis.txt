# Runtime Analysis

## Task0

### Python Operations Used
- Index operation, e.g `l[i]`
    - Complexity: O(1)
- Iteration, e.g `for v in l:`
    - Complexity: O(n)
    
### Complexity
```
Time Complexity: O(n)
Space Complexity: O(n)
```

## Task1

### Python Operations Used
- Iteration, e.g `for v in l:`
    - Complexity: O(n)
- Containment operation, e.g `x in/not in l`
    - Complexity: O(n)
- Index operation, e.g `l[i]`
    - Complexity: O(1)
- Append operation, e.g `l.append(5)`
    - Complexity: O(1)
